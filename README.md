Python-Port-Scanner

A Python-based Network Port Scanner built using Python and powered by Nmap for efficient network reconnaissance.  This tool performs fast port scanning on the most common ports of a target host and displays results with clean colored terminal output. It is designed as a simple and effective network reconnaissance tool for cybersecurity learners.

Features

✅ Fast scanning of top common ports
✅ Clean colored terminal output

🟢 Green = Open

⚪ White = Closed

🟡 Yellow = Filtered

✅ Displays service names running on open ports
✅ Lightweight and VM-friendly (works well on Kali Linux)
✅ Simple and easy-to-use command-line interface
✅ Ideal for learning network reconnaissance and penetration testing basics

Technologies Used:

Python
Nmap
colorama (for colored output)
pyfiglet (for CLI banner)
